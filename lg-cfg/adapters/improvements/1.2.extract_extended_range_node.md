# Extract `ExtendedRangeNode` Class

## Problem

Class `ExtendedRangeNode` (duck-typed node with extended byte range) is duplicated in 3 files with identical implementation.

## Affected Files

- `lg/adapters/langs/typescript/code_profiles.py`: lines ~280-300
- `lg/adapters/langs/javascript/code_profiles.py`: lines ~280-300
- `lg/adapters/langs/rust/code_profiles.py`: lines ~120-140

## Solution

Add to `lg/adapters/shared/utils.py`:

```python
from lg.adapters.tree_sitter_support import Node


class ExtendedRangeNode:
    """
    Duck-typed node with extended byte range.

    Used when element range needs to include trailing punctuation
    (semicolons, commas) that are separate nodes in the AST.
    """

    def __init__(self, start_node: Node, end_node: Node):
        self._original = start_node
        self.start_byte = start_node.start_byte
        self.end_byte = end_node.end_byte
        self.start_point = start_node.start_point
        self.end_point = end_node.end_point
        self.type = start_node.type
        self.parent = start_node.parent
        self.children = getattr(start_node, 'children', [])
        self.text = getattr(start_node, 'text', None)

    def child_by_field_name(self, name: str):
        """Delegate to original node."""
        return self._original.child_by_field_name(name)
```

## Migration

Replace in each file:

```python
# Before (in typescript/code_profiles.py)
def _create_extended_range_node(original_node: Node, semicolon_node: Node) -> Node:
    class ExtendedRangeNode:
        """Duck-typed node with extended byte range."""
        def __init__(self, start_node: Node, end_node: Node):
            self.start_byte = start_node.start_byte
            # ... 15 more lines
    return ExtendedRangeNode(original_node, semicolon_node)

# After
from lg.adapters.shared.utils import ExtendedRangeNode

def _create_extended_range_node(original_node: Node, semicolon_node: Node) -> Node:
    return ExtendedRangeNode(original_node, semicolon_node)
```

## Alternative: Inline Factory

If keeping separate `_create_extended_range_node` functions feels redundant, can also use directly:

```python
# In _extend_range_for_semicolon:
from lg.adapters.shared.utils import ExtendedRangeNode

def _extend_range_for_semicolon(node: Node, element_type: str, doc: TreeSitterDocument) -> Node:
    # ... logic ...
    if should_extend:
        return ExtendedRangeNode(node, next_sibling)
    return node
```
