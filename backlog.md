# Глобальный Backlog Listing Generator

## Небольшие фичи

- Убирается повторяемая часть пути из статистики и листинга
- **Режим `changes`** включать через сами шаблоны контекстов

## Верхнеуровневые крупные функциональные блоки

- Аддон Listing Generator для JetBrains IDEs
- Встраивание контекста в VS Code чаты (Copilot и Cursor)
- Сокращение кода через многоточие (оставлять только сигнатуру)
- Возможность дешевой агентной моделью подготавливать контекст (фильтры и сокращения) для дорогой модели
