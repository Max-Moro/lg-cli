# Глобальный Backlog Listing Generator

## Небольшие фичи

- Убирается повторяемая часть пути из статистики и листинга
- **Режим `changes`** включать через сами шаблоны контекстов
- Возможность сразу делать контексту доступными для ручного редактирования (опция в UI)
- Doctor переименовать в diag и перевести на WebView


## Верхнеуровневые крупные функциональные блоки

- Новая структура `lg-cfg`
- Аддон Listing Generator для JetBrains IDEs
- Встраивание контекста в VS Code чаты (Copilot и Cursor)
- Сокращение кода через многоточие (оставлять только сигнатуру)
- Возможность дешевой агентной моделью подготавливать контекст (фильтры и сокращения) для дорогой модели


## Новая структура `lg-cfg/`
- sections.yaml - список основных секций
- tests.sec.yaml - любые дополнительные секции
- common.ctx.yaml - корневой контекст
- rules.tpl.yaml - вставляемый шаблон для контекста
- sub-fold/ - любые поддиректории с секциями и шаблонами

Самих директорий `lg-cfg/` может быть несколько и они могут быть расположены на любом уровне.
